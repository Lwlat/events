# LwLAT Events Changelog

## Version 1.1.0
Released: 10 July 2026

### Added
- Large featured event card.
- Cleaner image-led event tiles.
- Responsive mobile layout.
- Branded LwLAT conference backdrop.
- White header text.
- Transparent PNG LwLAT logo.
- Temporary admin code: 1234.
- GitHub-backed media manager structure.
- Data-driven event configuration.

### Changed
- Tagline updated to: Winning hearts and inspiring minds to learn without limits
- Removed the Trust logo from event tiles.
- Improved card typography, spacing and hover behaviour.

### Known limitations
- Admin code 1234 is only a temporary client-side barrier.
- Automatic event capacity management is not yet implemented.
- Delegate editing is not yet connected to a central database.
- Netlify CMS requires GitHub, Netlify Identity and Git Gateway setup.
