LwLAT EVENTS: GITHUB + WEB MEDIA MANAGER SETUP

WHAT THIS PACKAGE ADDS
- /admin/ web editor
- Image upload library
- EduHub 2027 event editor
- Platform settings editor
- Git-backed version history
- Automatic Netlify redeploys after publishing

IMPORTANT
The admin editor requires the site to be connected to a GitHub repository.
A drag-and-drop-only Netlify site cannot save CMS changes permanently.

STEP 1: CREATE A GITHUB REPOSITORY
1. Sign in to GitHub.
2. Create a new private repository, for example:
   lwlat-events
3. Extract this zip on your computer.
4. Upload all files from inside the extracted folder to the repository root.
5. Commit the files to the main branch.

STEP 2: CONNECT THE REPOSITORY TO NETLIFY
1. In Netlify, select Add new project.
2. Choose Import an existing project.
3. Choose GitHub.
4. Select the lwlat-events repository.
5. Build command: leave blank.
6. Publish directory: .
7. Deploy the project.

STEP 3: ENABLE NETLIFY IDENTITY
1. Open the project in Netlify.
2. Go to Project configuration.
3. Open Identity.
4. Enable Identity.
5. Set registration to Invite only.

STEP 4: ENABLE GIT GATEWAY
1. In the Identity settings, find Services.
2. Enable Git Gateway.
3. Confirm that it uses the connected GitHub repository.

STEP 5: INVITE YOURSELF
1. Go to Identity.
2. Select Invite users.
3. Enter your email address.
4. Open the invitation email.
5. Set your password.

STEP 6: OPEN THE EVENT MANAGER
Go to:
https://YOUR-NETLIFY-SITE.netlify.app/admin/

From there you can:
- Replace the EduHub card image.
- Replace the EduHub hero image.
- Upload an event logo.
- Edit event wording, date, venue and capacity.
- Edit lunch and session choices.
- Upload general media items.

WHERE IMAGES ARE SAVED
Uploaded images are committed to:
media/uploads/

The event JSON is updated with the image path automatically.

PUBLISHING
Click Publish in the editor.
Decap CMS commits the change to GitHub.
Netlify detects the commit and redeploys the site.

SECURITY
Keep Netlify Identity set to Invite only.
Do not enable open public registration for the CMS.
